# Shopper Spectrum Project

## Overview
Shopper Spectrum is a Machine Learning project that performs:

1. Customer Segmentation using Gaussian Mixture Model (GMM)
2. Product Recommendation using Collaborative Filtering and Cosine Similarity

## Features
- Customer segmentation based on RFM values
- High-Value, Regular, Occasional, and At-Risk customer classification
- Product recommendation system
- Streamlit web application

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

- shopper_spectrum.py
- best_gmm_model.pkl
